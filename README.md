# FoodCo Arulogun — Business Operations Dashboard

A full-stack business operations dashboard built for a food service company. Provides real-time visibility into sales, expenses, inventory, and daily reporting.

## Live Demo
[foodco-arulogun.vercel.app](https://foodco-arulogun.vercel.app)

## Features
- Sales tracking and revenue reporting
- Expense management and cost breakdown
- Inventory monitoring
- Responsive dashboard with data visualization
- Role-based access control

## Tech Stack
- **Frontend:** React, Ant Design, TypeScript
- **Backend:** Node.js
- **Deployment:** Vercel

## Screenshots
![Dashboard](https://foodco-arulogun.vercel.app)

## Getting Started
```bash
npm install
npm run dev
```
